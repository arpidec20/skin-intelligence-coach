# Skin Intelligence Coach 🌿

An interactive AI skincare consultation platform — understand your skin, pick the right actives, and build a personalised, conflict-checked routine.

**Live site:** https://arpidec20.github.io/skin-intelligence-coach/

## Features
- 8-step skin profile assessment (skin type, concerns, sensitivity, life stage, climate, budget)
- Clickable face-zone map (9 zones) with zone-specific analysis
- 14 active ingredients explained in depth
- Ingredient conflict checker with weekly rotation calendars
- Personalised AM/PM routine builder with 43 ranked product recommendations
- Progress tracker with photos, trend charts and JSON export
- Weekly AI coach that adapts your routine
